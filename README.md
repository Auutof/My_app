# To-Do App This project is an Android To-Do application developed as part of a university assignment.
It allows users to add tasks and manage them using radio buttons. 
## Features
- Add tasks 
- Change task status 
- Simple UI 
## Technologies 
- Kotlin 
- XML 
- Android Studio
